# Security Policy

## Supported Versions
This repository follows a rolling support policy on the `main` branch.
Only the latest version on `main` is supported with security fixes.

## Reporting a Vulnerability
Please report security issues **privately** via GitHub Security Advisories:

- Go to: **Security → Advisories → New draft security advisory**
- Provide: description, impact, reproduction steps, affected versions, and any suggested fix.

We aim to:
- Acknowledge receipt within **2 business days**
- Provide an initial triage within **7 days**

## Coordinated Disclosure
Please do not open public issues for security vulnerabilities. We will coordinate disclosure and release notes once a fix is available.
